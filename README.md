# Lemina.

Next.js app for African tech market intelligence.

## Setup

```bash
npm install
cp .env.example .env.local
# add supabase credentials to .env.local
npm run dev
```

## Structure

- `/` - landing page
- `/founders` - company submission form  
- `/app` - investor dashboard (protected)
- `/api` - form submission and data endpoints

## Deploy

Vercel with environment variables.
